# 演示视频脚本（2–3 分钟）

## 0:00–0:15 开场

**画面**：产品 Logo + 标题「未音·时刻音乐盲盒」

**旁白**：
> 未音教会了每个人创作音乐。时刻盲盒，教会音乐来回应每个人。

## 0:15–0:45 Step 1 · 写+拍

**画面**：输入「今天下雨，但吃到了一碗很好吃的面。」+ 可选上传一张面/雨夜照片

**旁白**：
> 不必写情绪词，一句日常就够。照片提供氛围，文字解释意义，AI 更少误判。

## 0:45–1:05 Step 2 · 抽

**画面**：点击「抽取」→ 上滑手势 → 三个盲盒卡片，仅显示情绪文案

**旁白**：
> 系统生成三个回应。用户不知道背后是同频、转场还是偶遇，只选最像此刻的一句。

## 1:05–1:45 Step 3 · 看+听

**画面**：选择一个盲盒 → 开盒动画 → 开盒文案 → 播放 → 歌名 reveal → 视觉卡片

**旁白**：
> 开盒后才是完整体验：文案、歌曲、视觉卡片。歌名不会提前剧透，像真正的盲盒。

## 1:45–2:05 沉淀与分享

**画面**：收藏到情绪日历 → 导出 9:16 分享图

**旁白**：
> 听完可以收藏到情绪音乐日历，或保存一张专属视觉卡。

## 2:05–2:30 未音平台价值

**画面**：产品架构图 + 未音 App Tab 嵌入示意

**旁白**：
> 这是未音从创作工具到情绪产品的延伸。Demo 用预生成曲库；正式版可内嵌未音生成能力。

## 2:30–2:45 结尾

**旁白**：
> 未音·时刻音乐盲盒。把今天的一个瞬间，交给音乐。

---

## 录制建议

1. 手机竖屏录制 PWA（Chrome → 添加到主屏幕）
2. 准备 3 组输入：下雨+面 / 汇报不开心 / 地铁像旧友
3. 每组展示三盲盒差异 + 开盒 reveal
4. 结尾展示 `docs/product-spec.md` 中的架构图页
