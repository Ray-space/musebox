# 未音作歌 Prompt 记录

> 在未音 App / PC（https://y.qq.com/vemus/）中使用以下 prompt 批量生成 demo 歌曲。
> 生成后导出 MP3 到 `content/audio/`，文件名与 `songs.json` 中 `id` 一致。

## 场景 1：下雨 + 好吃的面

| 策略 | 未音模式 | Prompt |
|------|---------|--------|
| 同频 | 填词作歌 | 中文流行慢板，歌词围绕「雨夜小确幸」，情绪真实不煽情，像坐在便利店窗边，不承诺会变好，只是如实记录此刻 |
| 转场 | 灵感作歌 | 同上主题，副歌加入「热气升起」明亮意象，编曲从中速偏沉过渡到轻快节奏，情绪从安静到微微上扬 |
| 偶遇 | 图片/灵感 | 城市 Lo-fi 电子，用「湿漉漉的街/冒热气的面」做意象，曲风意外但画面一致 |

|------|---------|--------|

## 导出清单

1. 每首歌导出 MP3，命名 `{id}.mp3`
2. 放入 `content/audio/`
3. 确认 `songs.json` 中 `audioUrl` 路径一致
4. 选 1 首代表「盲盒理念」的歌曲投稿未音比赛
