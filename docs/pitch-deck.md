## 答辩 Q&A 预备

**Q：为什么没有实时调用未音 API？**
A：未音暂未开放公开 API。Demo 用预生成曲库验证体验；正式版需内嵌未音生成链路，我们已在方案中设计集成路径。

**Q：三盲盒如何避免听起来一样？**
A：策略分桶 + 不同 genre/tempo prompt + 用户在前台选择的是「感受」而非「风格」。

**Q：与 QQ 音乐推荐有何不同？**
A：推荐是「猜你喜欢」；盲盒是「回应你的此刻」，且有开盒仪式与视觉卡片沉淀。
